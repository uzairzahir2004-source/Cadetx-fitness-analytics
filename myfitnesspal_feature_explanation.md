# Task 05: Feature Engineering Explanation

## Engineered Features Overview
* **engagement_ratio:** Calculated as interaction depth relative to base exposure. Higher values identify highly active community spaces.
* **influencer_weight:** A scaled multiplier representing the authority score of the content creator or post author.
* **trend_velocity:** Measures the numeric acceleration or deceleration speed of a specific fitness metric or brand topic over a standard time window.

## Why These Matter
These transformed columns turn raw social text data into highly structured numeric features, which allow machine learning and predictive models to accurately project community trends and user retention behaviors.
